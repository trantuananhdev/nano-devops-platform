# HTTPS — `*.nano.platform`

Traefik terminates TLS with a dev wildcard certificate. All public apps use **HTTPS only** (HTTP → 301 redirect).

## One-time setup (VM)

```bash
./cli.sh certs
./cli.sh up
```

Trust the CA on your laptop (presenter):

```bash
vagrant ssh -c "cat /opt/platform/config/traefik/certs/rootCA.crt" > nano-rootCA.crt
# Import nano-rootCA.crt into OS/browser trust store
```

## Hosts file (presenter PC)

Point every hostname to the VM service IP (see `vagrant ssh -c "ip -4 addr show eth1"`):

```
<VM_IP> health.nano.platform data.nano.platform aggregator.nano.platform
<VM_IP> user.nano.platform faulty.nano.platform ai.nano.platform
<VM_IP> crm-demo.nano.platform
<VM_IP> crm-ingest.nano.platform
<VM_IP> grafana.nano.platform prometheus.nano.platform
```

## Certificate files

| Path | Role |
|------|------|
| `/opt/platform/config/traefik/certs/nano.platform.crt` | Wildcard server cert |
| `/opt/platform/config/traefik/certs/nano.platform.key` | Private key |
| `/opt/platform/config/traefik/certs/rootCA.crt` | Import on presenter laptop |

Generated by `platform/infra/scripts/system/generate_certs.sh`.
