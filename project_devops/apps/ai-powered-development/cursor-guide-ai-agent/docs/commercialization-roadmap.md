# Roadmap Thương mại hóa (bán được cho doanh nghiệp)

Mục tiêu của hệ không chỉ chạy demo, mà phải “đứng được trong vận hành”.

## MVP nên có (để bán được sớm)
- API nhận task + endpoint kiểm tra status
- Memory SSOT + audit log
- Task routing theo policy (không để AI làm bừa)
- Verification gate bắt buộc cho “final claim”
- Export report + log để human review

## Nâng cấp doanh nghiệp (để bán “đến cấp quốc tế”)
- Multi-tenant + RBAC/ABAC (ai được đọc/ghi dữ liệu nào)
- Incident/Failure mode catalog + lesson learned
- Integration layer cho enterprise systems (CRM/ERP/Ticketing)
- Connector framework (thêm tool mới mà không phá core)
- Benchmark / regression suite để giữ chất lượng qua phiên bản
